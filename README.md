## 类

### 修饰符

1. `public`: 所有人可见（默认）。
2. `private`: 只能被类本身调用，不能被类的实例调用，也不能被子类调用。
3. `protected`: 只能在类或类的子类中调用。
4. `readonly`: 只读。
5. `static`: 静态属性，可以被类或类的子类调用，不能被实例调用。
